Intoduction to Polkadot JS

Welcome to the tutorial series on Polkadot JS. In this video, we shall go over the outline of topics to be covered in this tutorial series and give an overview of what can be done using Polkadot JS. Let me quickly show you what you would see when you open the website. Let me highlight Polkadot JS Apps wallet which is a one stop shop for all the chains that run on Polkadot ecosystem. Let me navigate in and show you around what you would see. There you go! You have Polkadot, Kusama and Test networks along with their parachains listed. As you can see, this service is hosted on several networks powered by different entities. Along with providing access to wallets, this interface is capable of doing much more! We will learn about those details later on, but for now, let me navigate back to the homepage. Now, you see this card which says apps wallet ipfs! This is just a mirror of apps wallet which is hosted on dedicated servers. Why do we have this? Because one of the key things in Web3 vision is decentralization. IPFS powers the distributed web. So, when the service providers go down due to network issues or an attack, you can still access your wallets through this ipfs version.

Then we have the Polkadot JS extension that works on Chrome and Firefox browsers. As the name suggests, it is a vault that adds an additional layer of security to your account. How? We will discuss that in a dedicated video on wallets. Also, this browser extension will offer some protection against scams and phishing websites. Here, let me click on this link right here. There you go! Here is the list of websites that have been flagged by the community and the list keeps growing. If you open any of these websites, the browser extension will automatically detect them and prompt you to be cautious. If you are aware of any phishing websites, feel free to contribute to the list here!

The documentation is for the developers who like to interact with the Polkadot or Substrate chains and do other cool stuff using Javascript and finally the code of Polkadot JS is open and you can inspect, distribute, modify or otherwise use software for any purpose. As you may have noticed, Polkadot JS is a hub for so many features and functionalities, which can get quite overwhelming if you are a new user entering Polkadot ecosystem. You need to understand that the key audience for Polkadot JS website are the developers. But as a beginner, you can learn to perform basic tasks like creating an account, transferring tokens and perform staking through Polkadot JS apps and extension if you would like to. If you are looking for apps or websites that provide these basic functionalities, there are many out there that are trusted and vouched by the community. In this tutorial series, our main objective is to provide a concise introduction to all the key functionalities and features offered by Polkdot JS targeted for users who are new to the polkadot ecosystem.
